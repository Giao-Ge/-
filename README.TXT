实验器材:
	STM32+ESP8266+LORA开发板一块，型号为：S82
	
	
硬件资源:
	1,按键,wake_up(key1):PA0,key2:PB0, key3:PC5，
	2,灯,LED1:PA8，LED2：PC7，LED3：PC6
	3,25Q16，CS：PA4，S0：PA6，SI：PA7，CLK：PA5
	4,LCD(128*128)，SDI：PC13，SCL：PC0，RESET：PC1，DC：PB8，BLK：PC8
	5,蜂鸣器，PB1
	6,串口1，TX：PA9，RX：PA10
	7,串口2，TX：PA2，RX：PA3
	8,串口3，TX：PB10，RX：PA11
	9,ESP8266模块，接在串口2上，
	10,DS18B20:PB9
	11,空余的引脚：PA11,PA12,PA15
	　　　　　　　 PB3,PB4,PB6,PB7
	　　　　　　　 PC3,PC9,PC10,PC11,PC12
	
	
实验现象:
	LCD上正确显示温度和气压数据
  
注意事项:
	1,调试串口波特率为115200
